# Модальные окна

## Best practices
### Основные
- [ ] При открытии переносить фокус внутрь
- [ ] После закрытия возвращать фокус на кнопку, которая его открыла
- [ ] Зацикливать фокус внутри окна, не давая выходить за его границы
